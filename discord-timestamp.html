What is a Discord Timestamp?

If you run a Discord server, schedule gaming sessions, or manage an international community, time zones are a massive headache. If you announce an event at "5:00 PM EST," your members in London, Tokyo, and Sydney have to manually calculate what time that means for them.

A Discord Timestamp solves this problem completely. It is a special piece of markdown code using Unix Epoch time (for example, <t:1672531200:F>) that you can paste into any Discord chat or announcement channel. When you send the message, Discord automatically translates that code into the local timezone of whoever is reading it.

How to Use This Generator Tool

Our free Timestamp Generator makes it incredibly easy to format your dates without needing to calculate Unix time manually:

  • Step 1: Select the Date of your event from the calendar input above.
  • Step 2: Select the exact Time your event starts. The tool uses your current local timezone to do the math.
  • Step 3: Review the generated formats. You can choose from Short Time, Long Date, Relative Time (e.g., "in 5 hours"), and more.
  • Step 4: Click the "Copy Code" button next to your preferred style and paste it directly into Discord!

Available Timestamp Formats

Discord supports multiple visual styles for displaying time depending on the letter attached to the end of the code. Here is a breakdown of what the different format tags mean:

  • Default (t): Shows the short time (e.g., 4:20 PM)
  • Long Time (T): Includes seconds (e.g., 4:20:30 PM)
  • Short Date (d): Numbers only (e.g., 12/31/2026)
  • Long Date (D): Spelled out month (e.g., December 31, 2026)
  • Short Date/Time (f): Full text format (e.g., December 31, 2026 4:20 PM)
  • Long Date/Time (F): Includes the day of the week (e.g., Thursday, December 31, 2026 4:20 PM)
  • Relative (R): A dynamic countdown or count up (e.g., "in 3 days" or "2 hours ago")
